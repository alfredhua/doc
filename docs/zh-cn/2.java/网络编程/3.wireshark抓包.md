## Chrome 启动设置

Chrome 启动参数需要配置如下内容：

--user-data-dir 配置用户使用数据
--ssl-key-log-file 配置 ssl-key-log 的存储路径

```shell
$ /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --user-data-dir=/tmp/chrome --ssl-key-log-file=/tmp/.ssl-key.log

```

执行命令，会开启一个全新的 Chrome
