栈帧:每个栈帧对应一个被调用的方法，可以理解为一个方法的运行空间。

每个栈帧中包括

  • 局部变量表(Local Variables)
  • 操作数栈(Operand Stack)、
  • 指向运行时常量池的引用(A reference to the run-time constant pool)、
  • 方法返回地址(Return Address)和附加信息。
